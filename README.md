# Nenuphar

Visualize pressed keys for Linux

!!!THIS PROJECT IS IN DEV mode - not ready to use!!!

## Install

### from source

```shell
git clone https://github.com/tomaszkubacki/nenuphar.git
cd nenuphar
cargo build
```
